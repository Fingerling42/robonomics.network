Το προϊόν βασίζεται σε ένα σενάριο στο οποίο εργαζόμαστε πάνω από 7 χρόνια: έρευνα, πιλοτικές κυκλοφορίες και δοκιμές στον τομέα της πολιτικής παρακολούθησης της ποιότητας του αέρα στις πόλεις. Επιπλέον, θέλουμε να δείξουμε πώς ένα open source project, που υποστηρίζεται από μόλις χίλιους χρήστες, μπορεί να ανατρέψει ολόκληρη την αγορά επιτήρησης. Ονομάζουμε αυτό το project το Altruist open dataset. Περισσότερες λεπτομέρειες για την καμπάνια kick-start και την ιδέα του open dataset βασισμένου στους 'Altruists' — σε αυτήν την ανάρτηση.

## Βασικά οφέλη του Altruist [Urban & Insight]

Μοναδικά χαρακτηριστικά στην κατηγορία της συσκευής:   
- Σχεδιασμός διπλής μονάδας — για εξωτερική και εσωτερική χρήση   
- Ενισχυμένος αισθητήρας σωματιδίων — SDS011, κατάλληλος για αστικές συνθήκες   
- Αισθητήρας θορύβου εξωτερικού τύπου για παρακολούθηση της ηχορύπανσης   
- Αισθητήρας βαρομετρικής πίεσης — χρήσιμος για ορεινές περιοχές και όχι μόνο   
- Σύγκριση θερμοκρασίας και υγρασίας μέσα και έξω, συν αισθητήρας CO₂ στο υπνοδωμάτιο   
- Ευκαιρία να συμβάλετε σε μια **ανοιχτή τράπεζα δεδομένων** χρήσιμη για πανεπιστήμια και κοινωνικούς οργανισμούς   
   
**Χαρακτηριστικά ανοιχτού υλικού:**

- Τοπική διεπαφή IP με πλήρη έλεγχο της συσκευής   
- Πλήρης τοπικός έλεγχος των δεδομένων σας - χωρίς υποχρεωτικό cloud   
- Θήκη τρισδιάστατης εκτύπωσης - 9 επιλογές σχεδίου, ή αναπαραγάγετε τη θήκη στον 3D εκτυπωτή του φίλου σας χρησιμοποιώντας έτοιμα αρχεία STL στο GitHub   
- Προαιρετική συνδρομή σε ανοιχτό cloud (για απομακρυσμένο backup και διαμοιρασμό δεδομένων με οικογένεια, φίλους και κοινότητα)   
- Ενσωμάτωση με Home Assistant και έτοιμα πρότυπα αυτοματισμού   
- Ενσωμάτωση με ανοιχτούς χάρτες αισθητήρων   
- 3xO: πλήρως ανοιχτό firmware και ανοιχτό υλικό βασισμένο σε αρχιτεκτονική RISC-V   
- Θύρα USB-C για ανάπτυξη και ενημερώσεις firmware   
- Τεράστιες δυνατότητες DIY: συνδέσεις GPIO, δίαυλος I²C για επιπλέον αισθητήρες, firmware και διαγράμματα δημοσιευμένα στο GitHub!   
- Ο μικροελεγκτής RISC-V ESP-C3 υποστηρίζει επίσης Zigbee/Thread σε επίπεδο υλικού

## Αλτρουισμός με μακροπρόθεσμη πρακτική αξία!

Μαζί με την Altruist, αναπτύξαμε την υπηρεσία **sensors.social** — ένα open source cloud για αξιόπιστη κατανεμημένη αποθήκευση δεδομένων αισθητήρων χρηστών.

Αυτό είναι απολύτως προαιρετικό, αλλά αν επιλέξετε να συμμετάσχετε, τα δεδομένα από τον εξωτερικό σας αισθητήρα θα προστεθούν στο **Altruist Open Data Set for Universities and NGOs** για τα επόμενα 3 χρόνια.   
Η εμπειρία της ομάδας μας δείχνει ότι μόλις **15 έως 30 Altruists** αρκούν για να δημιουργήσουν ένα χρήσιμο dataset για μια πόλη με πάνω από ένα εκατομμύριο κατοίκους.

Χάρη στη μοναδική αρχιτεκτονική του **sensors.social** και της συσκευής Altruist, αυτό το dataset θα συσσωρεύσει πολύτιμα δεδομένα, καθιστώντας τους Altruists βασική πηγή δεδομένων για την ποιότητα του αστικού περιβάλλοντος.
