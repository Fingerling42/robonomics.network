## 认识产品本身——Altruist 空气质量套装：Urban & Insight

一款适用于室内外的智能空气监测仪：可测量城市空气污染物（PM）、室外噪音、卧室二氧化碳，并对比室内外温湿度。配备气压传感器，助力天气预测。E-Ink 显示屏、Wi-Fi、USB-C，无强制云端，完全兼容 Home Assistant，支持 DIY 和改装。

## Altruist [Urban & Insight] 的主要优势

同类设备中的独特功能：   
- 双模块设计——适用于室内和室外   
- 增强型颗粒物传感器——SDS011，非常适合城市环境   
- 户外级噪音传感器，用于监测噪音污染   
- 气压传感器——适用于山区及其他地区   
- 室内外温湿度对比，以及卧室内的 CO₂ 传感器   
- 有机会为大学和市民组织有用的**开放数据银行**做出贡献   
   
**开放硬件特性：**

- 本地 IP 接口，全面控制设备   
- 完全本地控制您的数据——无需强制云端   
- 3D 打印外壳——9 种设计选项，或使用 GitHub 上的现成 STL 文件在朋友的 3D 打印机上复制外壳   
- 可选开源云订阅（用于远程备份和与家人、朋友、社区共享数据）   
- 支持 Home Assistant 集成和现成的自动化模板   
- 可集成开放传感器地图   
- 3xO：基于 RISC-V 架构的完全开放固件和开放硬件   
- USB-C 接口，便于固件开发和更新   
- 巨大的 DIY 潜力：GPIO 接口、I²C 总线可扩展额外传感器，固件和原理图已发布在 GitHub！   
- RISC-V ESP-C3 微控制器还在硬件层面支持 Zigbee/Thread

## 兼具长期实用价值的利他主义！

我们与 Altruist 合作开发了 **sensors.social** 服务 —— 一个用于用户传感器数据可信分布式存储的开源云平台。

这完全是自愿的，但如果你选择参与，你的户外传感器数据将在未来3年内被加入到 **Altruist 高校与非营利组织开放数据集**。   
我们的团队经验表明，仅需 **15到30位 Altruist 用户**，就足以为一个拥有百万居民的城市创建有用的数据集。

得益于 **sensors.social** 和 Altruist 设备的独特架构，这一数据集将积累宝贵的资料，使 Altruist 成为城市环境质量数据的重要来源。
