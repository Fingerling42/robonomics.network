Robonomics 团队在曼谷展示了他们的首批设备阵容，**[“2025 年赛博朋克硬件”](https://x.com/AIRA_Robonomics/status/1856724439439913110)**。其中一款亮点设备是 Hikikomori Tamagotchi——这是一个极佳的教育项目！通过学习 Hikikomori 的运行方式，用户可以了解如何用不到 10 美元的设备直接向 Web3 云发送交易。结合 Robonomics 或 Vara / Gear.exe 的交互示例，用户还能获得管理设备数字孪生的实践经验。

从明年开始，**Gear Protocol** 将把 Tamagotchi 示例集成到 **[Gear.exe](https://gear-tech.io/gear-exe)——一款为以太坊 dApp 提供强大支持的无桥解决方案**，并将在全球范围内的**大学工作坊课程**中使用 Hikikomori 设备！

## Hikikomori 概述

Hikikomori 项目结合了软硬件，作为一个与智能家居系统集成的数字宠物，类似于 Tamagotchi，但又有所不同。它不再是传统的游戏玩法，而是通过 Home Assistant 与您的智能家居系统连接，每一次与家居的互动或自动化操作都会提升 Hikikomori 的“幸福感”水平。这个数字宠物还与 Robonomics 和 Vara 网络相连，使其在 Web 生态系统中拥有独特的“灵魂”。Hikikomori 会定期在 Robonomics 上更新其数字状态，示例代码可在[这里](https://github.com/airalab/hikikomori-tamagotchi/tree/only-robonomics/main)找到。

## 与 Vara Network 的集成

**通过利用 [Vara Network](https://vara.network)**——一个具备并行数据处理能力、实现深度可扩展性的 Layer 1 区块链——**Hikikomori 成为了一款专属的 Tamagotchi**，**其幸福感水平的计算逻辑**将在智能合约内完成。用于幸福感追踪的 Tamagotchi 智能合约可在[这里](https://idea.gear-tech.io/programs/0x8e5f2de1fea16db5a65d4e64bca1f8a709585853749b3572ff15487db2146771?node=wss%3A%2F%2Ftestnet.vara.network)查看。

## 在 VARA 上的测试与交易

代码已通过基础场景测试，即 Hikikomori 设备向 Vara 网络发送消息以提升其幸福感水平。该场景及相关代码示例可在[这里](https://github.com/airalab/hikikomori-tamagotchi/tree/main/main)查看。你还可以在 Polkadot.js Portal 上查看测试交易输出：
